### 三句话概括许文杰体系

> **1. 节点为王。** 择时第一——冰点转折上重仓，混沌期轻仓试，退潮期空仓等。
> **2. 只做最强。** 选股第二——横向对比找辨识度最高的，不做后排（杂毛），不格局（年票）。
> **3. 干净纯粹。** 执行第三——模式内买、模式内卖，不纠结不后悔，卖得明明白白。

## 每日交易决策速查表 ⚡ [v33 new]

**盘前 9:00 必做：**

### 一、定环境（决定能不能做）

| 问题 | 看什么 | 判断标准 |
|------|--------|---------|
| 今天是上升/混沌/退潮？ | 昨日高标反馈 + 跌停家数 | 高标继续强+跌停<5→上升；高标分歧+跌停5-15→混沌；高标跌停+跌停>15→退潮 |
| 量能如何？ | 两市成交额 | >1.8万亿→活跃；1.5-1.8万亿→正常；<1.5万亿→收缩 |
| 指数位置？ | 上证/创业板日线 | 站上5日线→安全；跌破5日线→谨慎；连续下跌→等冰点 |

**结论：** 
- 上升期+量能活跃 → 可做，重仓（7-9成）
- 混沌期+量能正常 → 可做，中等仓（3-5成）
- 退潮期或量能收缩 → 休息或轻仓试错（0-2成）

### 二、定方向（决定做什么）

| 看什么 | 判断 | 
|--------|------|
| 昨日最强板块 | 今天能否续强？→ 看龙头竞价 |
| 昨日分歧板块 | 今天有没有修复？→ 看前排是否反包 |
| 新题材 | 有没有新消息/新首板潮？→ 关注新方向 |
| 风格倾向 | 趋势还是情绪？→ 看昨日趋势股/情绪股表现对比 |

**选择：** 选昨日最强+今天续强，或者新方向首日。不做后排、不做杂毛。

### 三、定个股（决定买什么）

| 问题 | 答案 |
|------|------|
| 这票是板块里的第几梯队？ | 只做第一梯队（尾盘没炸/竞价最强/最主动） |
| 有没有辨识度？ | 是不是板块里大家都知道的那几只？ |
| 量能够不够？ | 竞价量达昨日成交4%以上→关注 |
| 有没有弱转强信号？ | 超预期高开+缩量封板→最优 |

### 四、定仓位（决定买多少）

| 情景 | 仓位 | 理由 |
|------|------|------|
| 冰点转折+共振向上 | 7-9成 | 最佳窗口，重仓出击 |
| 上升期延续 | 5-7成 | 顺势而为 |
| 混沌期分歧充分后 | 2-5成 | 小仓试错 |
| 退潮期 | 0-2成或空仓 | 防守第一 |

### 五、定卖点（决定什么时候走）

| 情况 | 操作 |
|------|------|
| 走弱（破均线/绿盘/落后板块） | 走 |
| 主动走弱（带头跌） | 立即走 |
| 被动走弱（跟跌但板块还强） | 可等一等 |
| 板块退潮/主线切换 | 走 |
| 炸板不回封 | 走 |
| 模式内卖点触发 | 严格执行，不犹豫 |

> **许文杰原话：**「模式内的卖点不能保证你一定卖在最高，但一定能保证你卖得明明白白。」

## 零、快速决策树

```
1. 情绪周期阶段？
   → 退潮/冰点 → 空仓或极小仓试错
   → 混沌期 → 控仓，只做最强
   → 上升期/修复 → 积极参与

2. 今天是不是节点？（情绪/指数/板块，至少一个成立）
   → 否 → 不出手

3. 风格对不对？（容量 vs 情绪 vs 并存）
   → 风格错配 → 放弃

4. 板块是主线还是轮动？
   → 非主线 → 放弃

5. 票在板块中排第几？（横向对比）
   → 后排 → 放弃

6. 买点对不对？
   → 弱转强（最佳）→ 分歧转移至（可打回封）→ 强更强/追高/接高潮 → 放弃
```

## 一、顶层框架：四位一体

分析市场的**唯一正确顺序**：

| 层级 | 内容 | 核心问题 |
|------|------|---------|
| **节点** | 情绪周期拐点 | 能不能做？（择时） |
| **面** | 指数、上涨家数、成交量、风格 | 环境支持吗？ |
| **线** | 板块轮动、主线方向 | 做什么方向？ |
| **点** | 高标、连板、核心股 | 具体做什么票？ |

> 节点不对，一切白费。个股尊重板块，板块尊重大势。风格也是一种大势。

### 指数共振体系

指数与情绪共振的四种状态：

| 状态 | 环境 | 操作 | 级别 |
|------|------|------|------|
| **指数+情绪共振向上** | 量能充足，两者同涨 | 重仓出击，做最强方向 | 主升浪 |
| **指数稳+情绪强** | 题材唱戏 | 做核心辨识度，前排龙头 | 黄金窗口 |
| **指数弱+情绪强** | 量缩但情绪独立 | 抱团行情，只做核心 | 中等 |
| **指数弱+情绪弱** | 双杀 | 空仓或极轻仓防守 | 冰点/退潮 |

**关键判断：** 指数好转≠情绪好转，必须有情绪正反馈（高标反包、跌停减少、上涨家数回升）才算真正的共振。

> **许文杰原话：**「指数是面，情绪是里子。面好看里子烂=不能做。面烂里子强=还能做抱团。面里都烂=休息。」

### 情绪与指数背离

当指数涨但核心个股背离（毒苹果判断）：不要被指数迷惑，看核心个股的反馈。

**背离实战案例：**
- 指数红盘但高标跌停→情绪退潮，指数是假象
- 指数绿盘但核心股反包→情绪复苏，可出手
- 50/300 拉指数但中证 1000 下跌→护盘而非真涨

> **许文杰原话：**「指数 OK 没问题，有消息刺激挺强的，下午也逐步回落了——为什么？因为情绪本身并不好。」

**判断优先级：** 情绪反馈 > 指数涨跌。指数只是个背景板，情绪才是能不能做的主因。

## 二、情绪周期

### 2.1 七个阶段

```
情绪冰点 → 修复 → 上升期 → 强分歧 → 高潮 → 退潮 → 混沌期
```

| 阶段 | 特征 | 操作 |
|------|------|------|
| **冰点** | 涨停<10，跌停>20，连板压缩到2板 | 轻仓试错前排，找低价+新题材 |
| **修复** | 指数阳线+共振板块，上涨家数回升 | 积极参与，做前排 |
| **上升期** | 连板高度持续拓展，一字龙+换手龙明确 | 大干特干，满仓单调 |
| **强分歧** | 高标批量炸板/断板，后排回落 | 减仓，观察谁扛住 |
| **高潮** | 批量涨停，情绪过热 | 卖出/减仓，不追 |
| **退潮** | 批量跌停，负反馈蔓延 | 空仓/控仓 |
| **混沌期** | 退潮后方向不明，肉面共存 | 控仓或空仓 |

### 2.2 周期位置速查

| 指标 | 冰点 | 修复 | 上升期 | 强分歧 | 高潮 | 退潮 |
|------|------|------|--------|--------|------|------|
| 上涨家数 | 极低 | 回升 | 高位 | 分化 | 极高 | 下降 |
| 涨停数 | <10 | 增多 | 爆发 | 减 | 批量 | 稀少 |
| 跌停数 | >20 | 减 | 极少 | 出现 | 无 | 增多 |
| 成交量 | 萎缩 | 放大 | 放大 | 缩/平 | 放大 | 缩量 |
| 连板高度 | 2板 | 突 | 持续 | 震 | 极值 | 崩塌 |
| 一字龙 | 无 | 出现 | 封单不减 | 可能开 | 封死 | 断板 |

### 2.3 赚钱效应 vs 亏钱效应

的首板有议价的对吧？然后华为是冲击了200，然后今天有一个泽成电子，他觉得泽成电子尾盘的这个强势回风勉强回封。泽成电子勉强回风是30厘米去卡10厘米。那我觉得不对，因为泽成电子它本身你看F10它是有PCB的。今天AR里面我刚才说过了，最强的分支是PCB对吧？那么PCB这个板块有足够的涨停板，有足够的赚钱效应，有足够的板块效应。  

### 2.4 一致性与分歧的循环

市场情绪在一致与分歧之间往复摆动，构成周期的底层节奏：

**一致→分歧（崩塌）：**
- 一致性预期过强→次日容易分化/反杀
- 标志：全网吹票、所有人看多、封单巨大但次日没人接
- "一致性预期太强，保不准最后一天给你反杀"

**分歧→一致（重生）：**
- 充分分歧后，市场筛选出最强方向，形成新共识
- 标志：分歧日扛住的票次日继续强

**分歧不够→反复拉扯：**
- 一字未开、分歧不充分→修复就是假修复→还会再砸
- "一字未开，分歧不够，就这一句话"

> **许文杰原话：**「交易的本质不在于预判对错，而在于应对。我主观倾向于混沌，但是否能走上升，看市场反馈。」

### 2.5 分歧分层

| 类型 | 表现 | 对策 |
|------|------|------|
| **局部分歧** | 仅高位震荡，低位有序 | 汰弱留强 |
| **整体分歧** | 高位崩塌+低位无序 | 全面防守 |



### 2.6.1 退潮期不做弱转强 ⚡ [v31 new]

退潮环境下，弱转强信号是陷阱而非机会。真正的弱转强必须发生在非退潮期（混沌或上升期）：

> **许文杰原话：**「节点是择时，点线面是择股。节点=合适的时机。冰点也好、启动点也好、弱转强也都是节点。面=市场环境 OK 的时候，不是什么退潮期的时候——混沌期也好、上升期也好，都是非退潮期的时候。线=找到合理的主线，点=参与辨识度个股。」

**退潮期弱转强的风险：**
- 退潮期已有资金拿先手→次日弱转强过于明牌→兑现盘砸场
- 退潮三天后的修复是真博弈点（前一天尾盘低吸），而非次日拉升后追

> **许文杰原话：**「有朋友问金奥博这种能不能去？弱转强板能不能打？不太合适——太明了。昨天尾盘已经有先手去拿，不够好。你真要去博弈退潮三天之后的修复，不如昨天拿。」


### 2.6 退潮期专项

> 📎 参见: [2.6.1 退潮期不做弱转强], [14.19 亏钱两大源头], [12.4 红线禁忌]

**五大忌：** 不接加速、不抄底（退潮期反包大概率失败）、不做中位（跌最惨）、不幻想穿越、不等节点就出手。

**四种策略：**

| 策略 | 说明 |
|------|------|
| 防守为主 | 不亏钱 > 赚钱，控制回撤比进攻重要 100 倍 |
| ETF 低保 | 冰点+无载体可买→买 ETF 当底仓保底 |
| 小账户解痒 | 分两个账户，退潮期用小账户练手 |
| 可转债日内 | 不确定能否拿次日→日内撸转债 |

**退潮结束信号：** 涨停数回升、跌停<5家、成交量放大、新题材+板块效应、高度突破前高。

**⚠️ 退潮结束 ≠ 上升期开始：** 可走上升也可走混沌。判断——放量中阳+新主线→上升期；无量+无主线→混沌期。

**退潮确认信号链：** 高位断板→中位补跌→后排崩塌→冰点数据。

**万亿环境下的新退潮方式：** 连续三四天肉面共存后的一次性节点。不是集体崩塌，温和退（退一退返一返）。适应方法：积累样本、找规律。

### 2.7 牛熊退潮差异

**案例：** 2024.12 机器人退潮（牛市）=肉面共存，金奥博天地板但次日有人翘板；2023 年熊市退潮=集体 A 杀，无翘板。

| 维度 | 熊市退潮 | 牛市退潮 |
|------|---------|---------|
| 方式 | 集体崩塌，A杀 | 肉面共存，温和退 |
| 时长 | 2-3天 | 拉长，退一退返一返 |
| 活口 | 几乎无 | 有穿越品种 |

### 2.8 情绪冰点修复载体

冰点修复日，资金优先攻击的方向：
- 超跌老龙头（辨识度高，套牢盘轻）
- 新题材首板（新故事+低位）
- 次新股/近端次新（无套牢盘）

### 2.9 极端情绪形态 ⚡ [v27 new]

| 形态 | 定义 | 信号含义 |
|------|------|---------|
| **核按钮** | 竞价挂跌停价大单出逃 | 极度恐慌，A 杀预期 |
| **天地板** | 涨停→跌停（20cm 振幅） | 极度恐慌，次日大概率低开 |
| **翘板** | 跌停板被资金打开拉红 | 情绪转折信号，可能是反转 |

**核按钮判断：** 竞价核按钮 + 9:38 前不翘板 → 当日情绪大概率差。竞价有缓和（跌幅收窄、有大单接）→ 可能有修复。

**翘板 = 情绪转折锚定：** 9:40 前翘板 + 其他辨识度票同时正反馈 → 当日情绪回暖确立。"电光 9:38 翘板+建设中博超预期=情绪回温。"

**天地板 ± 首阴：** 天地板次日大概率低开，急杀后反包的=强修复信号。但如果次日继续跌停=退潮确认。

## 三、节点思维

### 3.1 多节点体系

节点不止一种，情绪割裂时切换：

| 节点类型 | 锚定点 | 触发条件 |
|---------|--------|---------|
| **情绪节点** | 情绪周期拐点 | 冰点转折/高位退潮确认 |
| **指数节点** | 放量长阳 | 指数放量+板块共振 |
| **板块节点** | 板块周期拐点 | 首分→次日刹车→确认 |

### 3.2 板块节点两步确认

```
第一步（左侧/潜在）：首次大分歧
  → 连续加速无分歧多日→一旦分歧可能走镜像

第二步（右侧/确认）：次日没有走镜像，反而修复
  → 超预期 → 确认节点成立
```



### 3.3.1 指数长阳日怼高标 ⚡ [v31 new]

市场普涨、指数探底回升长阳之日，是短线资金本性暴露的时刻——第一时间看向最有辨识度的高标：

> **许文杰原话：**「就像指数长阳日怼高标是很好想到的。因为当天晚上会爆吹，市场是低点了，然后最高标很容易走加速，很容易有其他资金来顶——就是这部分资金去套利。」

**逻辑链：** 长阳→晚间爆吹→次日高标加速→套利资金涌入。

**注意：** 此策略适用于节点日（情绪/指数冰点后的转折长阳日），不适用于震荡中的随机阳线。

### 3.3 启动节点 = 最佳开仓窗口

> 📎 参见: [3.3.1 指数长阳日怼高标], [每日决策速查表-定仓位], [12.1.1 仓位控管]

- 竞价冰点 + 大市冰点 + 开盘白织带预期 → 回踩是买点
- 新板块首板潮：必须上车前排，不追后排
- 聚焦最强方向，不铺多条线



### 3.4.1 普涨次日上午不接力 ⚡ [v31 new]

全市场普涨后的第二天上午，是最容易亏钱的窗口：

> **许文杰原话：**「普涨次日买点后置，早上肯定会有惯性冲高。不要早上去瞎追，更多的是下午分歧关注前排。」

**原理：**
- 普涨当天产生的获利盘，次日早上惯性冲高后兑现
- 上午追涨=接了昨日获利盘的货
- 正确姿势：等午后分歧释放完，再看前排谁扛住了

**区别：** 指数放量长阳日→普涨次日公式适用；小阳/缩量修复日→不适用普涨次日逻辑。

### 3.4 普返日公式

普返日（普涨修复日）的判断：小票跟上 + 上涨家数 4000+ + 指数阳线。

普返次日：顺势冲高是"顺昨天的事"→不追；等分歧后再看方向。

## 四、日内节奏

### 4.1 竞价阶段（9:15-9:25）

**竞价预期差：**

| 情况 | 判断 | 做法 |
|------|------|------|
| 竞价爆量高开（达昨成交 4%） | 潜在弱转强 | 关注 |
| 竞价低于预期 | 偏弱 | 谨慎 |
| 竞价超预期 | 强信号 | 可试错 |
| 竞价冰点+指数竞价低点 | 可能转折 | 找买点 |
| 竞价金额排名→判断当日主线 | — | — |

### 4.2 开盘阶段（9:30-10:00）

**白织带/黑织带：**

| 形态 | 定义 | 信号 |
|------|------|------|
| 白织带 | 开盘价=最低价（连小数都不差） | 极端单边上涨→有持续走高预期 |
| 黑织带 | 开盘价=最高价（连小数都不差） | 极端单边下跌→不太会 V 型反转 |

**弱转强量化：**
- 可以低开，不能太深
- 1 分钟内拉到 +5% 以上
- 5%+ 稳到收盘
- 竞价爆量达昨成交额 4% 左右

**10 点看量法则：** 10 点还不能放量→当天大概率回落。

### 4.3 两种次日路径

| 路径 | 走势 | 风险 |
|------|------|------|
| **先砸后修** | 开盘杀跌→盘中出冰点→修复 | ✅ 安全 |
| **先冲后杀** | 开盘顺势拉高→冲高回落 | ⚠️ 最坑人 |

> "先顺势冲高再分歧——最容易亏钱。先砸早盘反而容易有冰点和修复。"

**顺势陷阱：** 今日高潮→明日顺势冲高="顺昨天的事"，追顺势=接高潮。亏损主要来源之一。

### 4.4 盘中量价（10:00-14:30）

| 量价 | 含义 | 操作 |
|------|------|------|
| 放量上涨+共振 | 强 | 可追/持有 |
| 缩量上涨 | 锁仓 | 可持有 |
| 放量下跌 | 恐慌释放 | 可能见底 |
| 缩量下跌 | 无人接盘 | 还要跌 |
| 放量+滞涨 | ⚠️ | 偏震荡，控仓 |

**下午补量判断：** 下午量能没超上午预估量→拉高也持续不了。

### 4.5 尾盘阶段（14:30-15:00）

| 现象 | 判断 | 操作 |
|------|------|------|
| 尾盘炸板 | 弱 | 必走（退潮期尤其） |
| 涨停封不住 | 尾盘打开=弱 | 走 |
| 尾盘偷鸡板 | 偷尾盘 | 次日大概率无溢价 |

### 4.6 尾盘定调与次日预案

尾盘走势决定次日预期，是盘后复盘的第一步：

| 尾盘走势 | 次日预期 | 操作 |
|---------|---------|------|
| 尾盘走强（抢筹） | 次日顺势冲高 | 不追，等冲高回落再看 |
| 尾盘走弱（跳水/炸板） | 次日可能低开 | 可等冰点低吸 |
| 尾盘炸板回封 | 次日期待弱转强 | 观察竞价强弱 |
| 尾盘偷鸡板（尾盘拉涨停） | 次日大概率低开 | 不追，是套利不是主升 |

**尾盘抢筹的判断：**
- 14:55-15:00 竞价阶段有资金明显拉高→次日有预期
- 但「尾盘抢筹」不是必然次日涨——要看抢的是谁、为什么抢
- 特斯达 vs 颗粒：尾盘有资金抢 vs 没资金抢→特斯达更强

> **许文杰原话：**「特斯达尾盘还有资金在抢，很明显它更强。颗粒早上逢反弹，尾盘没有资金抢筹。我把两者比较，到底是托斯达更强还是颗粒更强？最后还是觉得特斯拉更强。」

### 4.8 分时流速 ⚡ [v31 new]

**分时流速** 是许文杰的实战盘口语言，指个股拉升时每笔大单的频率和密度：

> **许文杰原话：**「大千的盘口当中的流速不够——说白了就是大单不够、大单不够快。」

**流速快=强：** 每笔几百万的单子快速连续成交，说明大资金在积极买入
**流速慢=弱：** 挂单稀疏，小单就能推几个点——这种拉升不可持续

**实战判断：**
- 涨停炸板后流速快+量大→容易挂（真筹码出逃）
- 涨停封板后流速稳→筹码稳
- 拉升时流速低→拉升虚，次日无溢价

> **许文杰原话：**「苏杭它其实流速很低的，就一笔差不多 200 的单子就能给它顶个两三个点，流速很低，盘口其实比较空的。」

### 4.9 红盘时间法则 ⚡ [v31 new]

判断个股当日强势的关键维度：**维持红盘的时间长度**。

> **许文杰原话：**「红盘维系的时间段时长跟绿盘时长是一个关注点。红盘的时间段要足够长。我们说美日互动很强，成绩很好——它翻绿一秒钟都没有。」

**判断标准：**
- 几乎全天红盘（无绿盘）→ 最强
- 大部分时间红盘→ 还行
- 红绿各半→ 一般
- 全天仅短暂红盘→ 弱
- 全天无红盘→ 弱极

**应用场景：** 判断高位股是否扛得住分歧。10 天 100% 异动后，红盘时间足够长才能稳住筹码。

### 4.7 分时图战法 ⚡ [v30 new]

**分时图配合 K 线判断强弱：**

| 分时形态 | K线配合 | 含义 |
|---------|---------|------|
| 缩量板（分时量极小） | 接近前高 | 强——筹码锁定好 |
| 爆量板（分时量巨大） | 突破前高 | 换手充分，关键考验 |
| 冲高回落（尖尖） | 高点不过前高 | ⚠️ 盘口大单冲不上去→卖点 |
| 探底回升（V形） | 下影线 | 承接强，可能是止跌 |
| 低开高走 | 白织带 | 开盘承接强，当日偏多 |

**分时卖点口诀：** "一波拉高之后，后面这个高点不过前面高点，盘口上面明显冲不上去→S 掉。上面有几个大单冲不上去，100 S 走掉就很容易烂在一个尖尖。"

**分时图设置：** 许文杰建议自定义版面，K线+分时图+热点板块同屏，按自己喜好布局。

### 5.2 辨识度分级

| 级别 | 角色 | 特征 |
|------|------|------|
| **一字龙** | 最高 | 顶一字封单不减，板块风向标 |
| **换手龙** | 高 | 换手充分，真正合力品种 |
| **20cm高度** | 高 | 涨幅最大，率先翻倍 |
| **容量中军** | 中高 | 成交额大，板块稳定器 |
| **低位二板** | 中 | 高低切载体 |
| **首板前排** | 中低 | 板块首板潮当日 |

### 5.3 共振梯队

连板梯队的核心规律——梯队完整度决定板块生命力：

> **许文杰原话：**「二板是跟情绪共振的。首板日=拿先手，二板日=共振梯队（最多），D3+开始分化为应对日。」

**梯队完整度判断：**
- 首板→二板→三板→四板→高度板各层都有 → 板块健康
- 梯队断层（缺某层）→ 弱势，板块可能快结束了
- 二板最多=共振最强→说明资金认可这个方向

**共振梯队的意义：**
- 共振梯队（二板日）是板块「确认」的时刻
- 首板=试盘，二板=认可，三板以上=比拼辨识度
- 一个板块如果没有二板梯队→一日游

**实战：** 每天统计各板块的连板梯队完整度，二板最多的板块=当前主线。

### 5.4 连板分析

**一字龙 vs 换手龙：** 一字龙=绝对前排、封单判断板块强弱。换手龙=越换手越安全、真合力。

**连板梯队完整度：** 梯队完整（首板→二板→三板→高度板都有）=板块健康。梯队断层=弱势。

**身位优势：** 先涨停的票对后涨停有身位优势。同属性横向对比，先板的身位更高。

### 5.5 后排四梯队（持仓强弱判断）

| 梯队 | 标准 | 策略 |
|------|------|------|
| 第一梯队 | 尾盘没炸板 | 可看 |
| 第二梯队 | 炸过又回封 | 弱，优先减 |
| 第三梯队 | 炸过没回封 | 更弱，必走 |
| 第四梯队 | 连涨停都无 | 忽略 |

### 5.6 中位股动态定义

中位不是固定板数，相对市场总高度：总高 8 板→4-5 板算中位；总高 13 板→7-8 板算中位。

### 5.7 涨停板定性

| 类型 | 定义 | 强弱 |
|------|------|------|
| **主动涨停** | 率先拉板，不依赖板块/指数 | 强 |
| **被动/反推涨停** | 被后排涨停潮推上去 | 弱 |
| **烂板回封** | 盘中多次开板→尾盘回封 | 分歧大 |
| **一字板** | 封死全天 | 最强 |

### 5.8 连板进阶分类 ⚡ [v27 new]

**连板量价四种组合（2 日视角）：**

| 类型 | D1→D2 | 强弱 | 特征 |
|------|--------|------|------|
| 换手+换手 | 两日都换手 | ✅ 最强 | 真合力，走得远 |
| 一字+一字 | 两日都一字 | 🟡 强 | 买不到，博弈一字首开 |
| 换手+一字 | 换手后加速一字 | 🟡 转强 | 超预期 |
| **一字+换手** | 一字后开板 | ❌ **最弱** | 获利盘集中抛售 |

> "一字加换手是最弱的。你想一字加换手什么意思？前一天大家都买不到，一开板全抛了。" — 许文杰

**一字首开：** 连续一字后首次开板的票，是一种特定博弈模式。前提：市场整体情绪好+板块没退潮。如果一字首开能板且次日给溢价→说明该模式可行→更多资金参与。

**缩量加速：** 连板不放量一直缩量=走不远。"建设如果明天后天都缩量加速，走不远的，一定会爆量检验。"

**爆量检验：** 龙头必经的换手考验，爆量日（成交额显著放大）能扛住封板=强；爆量日高开低走长阴=终结。

## 六、板块分析

### 6.1 板块四阶段

```
萌芽（1-2只，观察）→ 加速（3-5只+一字龙，可试错前排）
→ 分歧（后排回落，做换手回封或活口）→ 回流（核心穿越，确定性最高）
```

### 6.2 新板块诞生条件

**必要：** 低位+新、首板爆发（板块效应）、有故事可讲、指数共振。

**加分：** 政策催化、容量中军、一字龙。

**资金喜新厌旧：** "老名词炒不动→等新名词包装同一逻辑→新名词=主线启动。"（AI应用→Sora/Kimi/豆包）

### 6.3 每日板块三步法

1. **列辨识度**：按角度（容量/10cm连板/20cm）列出所有辨识度
2. **横向对比**：同角度内对比找最强
3. **制定预案**：明天分歧还是加速→什么条件可买

### 6.4 板块最强分支确认

资金在板块内轮动试错→只有一个分支打出高度→该分支=最强。

前提：高度股打出高度+次日正反馈→看高一线。高度不行→下面都不行。

### 6.5 镜像原理

学是说外面好像很多都在看分歧，对吧？那如果说分歧一致性看分歧的一致性预期过强，那确实对于节奏上面会有一定的影响。就比如说你要么就是分起来的比较晚，每次分析都不充分，这是一种。要么就是说真的风险来临的时候，他不会立马结束，他可能给你个连续隔两天甚至三天，对吧？  

### 6.6 23 定律

许文杰交易体系中最重要的节奏规律之一——板块和情绪的涨跌有内在的数字节奏：

**核心规律：**
- **高潮两天 → 调整三天 → 修复**
- "连续涨两天后第三天大概率调整，调整两天后容易有修复"

**实战应用（三步判断法）：**

| 天数 | 阶段 | 操作 |
|------|------|------|
| D1 启动 | 首板潮/弱转强确认 | 可上车前排 |
| D2 发酵/高潮 | 板块全面扩散 | 持有，不加仓 |
| D3 分歧 | 前排加速+后排掉队 | 减仓后排 |
| D4-5 调整 | 板块整体回调 | 等调整充分 |
| D5-6 修复 | 调整两天后修复 | 关注前排谁先修复 |

**23 定律不是死公式：**
- 是概率规律不是100%——调整两天后大概率修复，但不一定是强修复
- 要做的是前排品种→规律性强
- 不做后排/中位→随机性大，"板块修复你个股可能还跌"

> **许文杰原话：**「调整两天之后容易有修复——不是100%，但是概率比较高的一个规律。三天两冰点了，第四天容易有修复，有些强修复，有些弱修复，但大概率会有修复。前排规律性会比较强，后排或中位没那么大规律。」

**结合 23 定律的每日判断：**
- 今天板块涨了第几天？→ 第 2 天就要谨慎
- 今天板块调了第几天？→ 第 2 天开始找低吸机会
- 板块刚启动 → 积极；已连续涨 3 天 → 等分歧


### 6.7 趋势板块高潮=尾声

趋势股沿均线慢涨→突然涨停潮→短线客看到才进场时=已尾声。高潮日=该卖。

### 6.8 板块降级信号链

板块降级信号链：涨停数减少→高度股断板→后排先崩→中军破位→主线降级为普通轮动。四分之三黄牌法则：板块3/4涨停炸板→降级定量警告。

### 6.9 预期差博弈

主线分歧日→找可能续强方向（如AI分歧→机器人会不会续强）→分歧时低吸续强方向。核心：不想预期差怎么做股票？要多想一步：今天强的明天会不会弱，今天弱的明天会不会强。



### 6.14 点线面体系 ⚡ [v31 new]

个股、板块、大环境的三层制约关系，是许文杰分析框架的底层逻辑：

> **许文杰原话：**「点尊重线，线尊重面。个股尊重板块，板块尊重大师。是在说一件事——都在说李先生。」

**三层体系：**
| 层级 | 含义 | 权重 |
|------|------|------|
| 点 | 个股自身的强弱 | 最低 |
| 线 | 所属板块的强弱 | 中等 |
| 面 | 市场整体环境（指数+情绪）| 最高 |

**实战应用：**
- 个股强但板块弱→双拳难敌四手，尊重板块
- 板块强但面（指数/情绪）弱→持续性存疑
- 板块+面都 OK→此时才看个股强弱选前排

> **许文杰原话（个股尊重板块）：**「医药板块在周五整体板块是弱的。你一个蔬菜自身强，双拳难敌四手，个股尊重板块，板块尊重大师，你得尊重板块。」

### 6.15 趋势/情绪跷跷板 ⚡ [v31 new]

趋势股与情绪股之间存在跷跷板效应，尤其在大票与小票之间：

> **许文杰原话：**「炒的不是板块之间的轮动跷跷板，炒的是每一个板块之间的个股抱团。看板块他们是互卡，看情绪他们其实是一家人。」

**跷跷板规律：**
- 趋势强→情绪弱（机构主导行情，游资休整）
- 情绪强→趋势弱（游资主导行情，机构票被抽血）
- 两者共振→主升浪（少见的黄金窗口）
- 两者共振弱→冰点/退潮

**识别时机：** "短期又有一段炒小炒差的行情了。机构类、容量类、趋势类的，短期要先规避一下。"

### 6.10 卡位 / 模仿 / 反推

- **卡位：** 同属性争日内地位→板块未成主线时互卡=走不远
- **模仿：** 同批次形态复制→模仿到极致后崩塌
- **情绪传导链：** 主板→创业板→北交所，逐级溢出

### 6.11 强区抱团路径

市场弱势区/震荡期，资金抱团最强辨识度个股形成强区。强区内的品种互相托举，弱区品种被抛弃。特征：板块无批量强区，但局部个股持续走强区。操作：只做这些抱团强区股，不做跟风。

### 6.12 锚定票开盘定方向

前天不确定板块方向→选最敏感品种为锚定→设预期（板块上=锚定必须弱转强）→开盘即知。

## 七、风格判断


### 6.13 大小票联动预案 ⚡ [v26 new]

当大票连续 2 天明显强于小票（如中际/新易盛/天孚维持强势 vs 小光批量滞涨黄牌），板块不再是简单的「强区应对」，需要做**两级预案**：

| 结果 | 触发条件 | 操作 |
|------|---------|------|
| **大带小起** | 大票继续新高，小票被感染止跌回升→化解滞涨 | 接着奏乐接着舞，继续强区跟踪 |
| **小拖大下** | 小票持续走弱，最终把大票也拖下水→板块整体弱化 | 转为防守，板块降级为轮动 |

> "大的依然已经连着两天了，大的比小的强，我们要看大小之间的联动情况了——到底大的让小的止跌捞起来，还是小的把大的拖下水。" — 5/14

**操作锚定：** 指数止跌时观察大票是否还能新高 + 小票滞涨是否化解。宁可等确认再跟，不提前博弈。

### 7.1 三种风格

**案例：** 2025.3 机器人=趋势行情（容量票主导）；2025.1 跨年=情绪行情（连板妖股）；2025.4 混沌期=抱团行情（3-5 只票来回做）。

| 成交量 | 主导风格 | 选股方向 |
|--------|---------|---------|
| 2 万亿+ | 容量风格 | 中军，成交额导向 |
| 1.5 万亿以下 | 情绪风格 | 连板小票 |
| 1.5-2 万亿 | 混合并存 | **最优** |

> "风格也是一种大势，要尊重。风格不对，个股再好也不做。"

### 7.2 容量 vs 情绪深入

- 容量风格：中军主导，沿均线走趋势，涨停少。纯容量环境"做起来有点随机性"。
- 情绪风格：连板主导，有规律可循。
- 并存：短线选手最自如。

**情绪 vs 趋势新定义：** 情绪=连板高度+涨停数；趋势=均线+成交额。两者可以并存但不一定共振。

### 7.3 10cm / 20cm / 30cm

- **10cm 错觉：** 涨两天 vs 20cm 涨一天，场外人觉得 10cm 更牛逼，涨幅相同。"长得慢→显得久。"
- **30cm 量能上限：** 首板 2-3 亿理想，>4 亿难接力→只能套利。

### 7.4 大票 vs 小票

小票态度决定市场氛围。中证 1000 跟上=真正修复。先大票搭台→小票唱戏。

**大小票联动法则：** 大票稳住搭台→小票活跃→回流传导到大票→良性循环。大票崩→小票难独善其身。

### 7.5 半路 vs 打板

打板是小资金的做法，半路/点火是大资金的做法。没到那个体量不用学点火。打板追求确定性（等封板最后一刻下单），半路追求成本优势（不等封板就介入）。许文杰建议：小资金老老实实打板，确定性比成本重要。

### 7.6 止跌震荡 vs 滞涨震荡

止跌震荡：跌不动了，底部横盘，可低吸。滞涨震荡：涨不动了，顶部横盘，警惕回落。判断依据：止跌=出现锤子线/阳包阴+量能稳定；滞涨=指数冲不动+黄牌增多+大票力度衰减。

### 7.7 三大路径优先级 ⚡ [v26 new]

在指数震荡期（非主升），许文杰定义了三大操作路径的优先级排序：

| 优先级 | 路径 | 适用环境 | 特征 |
|--------|------|---------|------|
| **1** | 强区抱团 | 指数震荡 + 量够 | 局部个股抱团走强区，板块无批量强区 |
| **2** | 板块路径 | 指数主升 + 量充沛 | 板块批量强区，大面积主升 |
| **3** | 情绪抱团 | 指数止跌 + 量不足 | 只做最核心辨识度，范围更小 |

> "指数在走上升期才有强区行情，指数走震荡期更多是强区抱团路径。板块路径还没那么快，倾向强区抱团。" — 5/19

**路径切换信号：** 量能跌破 3 万亿 + 大票黄牌增多 → 板块路径不可行 → 转强区抱团；指数止跌阳线 + 量未恢复 → 强区抱团仍可做。

### 8.3.1 锁筹：弱转强需要缩量 ⚡ [v31 new]

弱转强成功的核心密码：**缩量涨停=锁住筹码**。

> **许文杰原话：**「只有缩量才能把前一天不坚定的筹码锁住，叫锁筹。只有不放量、只有封死涨停，才能把前一天不坚定的筹码锁住。一旦封不住，前一天的资金涌出来，量能放出来，接不住。所以弱转强的当天需要缩量。」

**弱转强缩量逻辑：**
- 前一天放量→分歧大→很多人买了
- 这些「不坚定的筹码」次日会卖
- 唯有缩量涨停才能让他们「不卖」→锁筹成功
- 一旦放量→说明锁不住→接不住→弱转强失败

**实战口诀：** 真正的弱转强 = 缩量 + 封死 + 次日继续强

### 8.3 弱转强 vs 强更强

| 类型 | 定义 | 买点？ |
|------|------|--------|
| **弱转强** | 前日弱→今日超预期转强 | ✅ **最佳** |
| **强更强** | 前日已强→今日加速 | ❌ 不接 |

**分歧转移至（正确接力方式）：** 高开冲板→盘中炸开→回封。只能打这种板。

### 8.4 攻防一体平衡

"用性价比位置买入+仓位控制风险。不弱≠立马大买，还是要找性价比+仓位平衡。"

| 动作 | 说明 |
|------|------|
| 找性价比位置 | 不在 7-8 点追高名牌 |
| 仓位平衡 | 不满不空，等信号 |
| 动态应对 | 走不出就 S，回平台就 B |

### 8.5 止盈纪律

止盈比止损更难——因为涨的时候人会贪婪。许文杰的止盈体系：

**六大止盈场景：**

| 场景 | 信号 | 操作 |
|------|------|------|
| 一致性高潮 | 全网吹票、板块全板 | 边涨边减 |
| 高位滞涨 | 连涨后缩量横盘 | 减半仓 |
| 分歧转弱 | 板块分歧日自己先跌 | 全走 |
| 分歧末端开板 | 一字跌停在分歧末端打开 | 可等反弹但不加仓 |
| 首次分歧 | 上升期第一次分歧 | 可持有/做T |
| 二波乏力 | 二波冲不过前高 | 走 |

**止盈的核心原则：**
- 不要追求卖在最高——卖在模式内的卖点就行
- 涨了不等于会继续涨——"亢奋的时候提醒自己，说不定是卖点"
- 有利润垫≠可以无限格局——核心票可多格局半天，后排票该走就走

> **许文杰原话：**「模式内的卖点不能保证你一定卖在最高，但一定能保证你卖得明明白白。不年票不亏大钱。」

**止盈后的心态：**
- 卖飞了不要后悔——"卖飞说明你判断对了方向，下次再来"
- 不要卖完又追回来——尤其是建设工业那种坑人的


### 8.6 做 T 心法

趋势股持有期间做 T：均线附近低吸→拉高 T 出，降低成本。不贪，有 2-3 个点就满足。

### 8.7 打板操作术语 ⚡ [v27 new]

**扫板 vs 排板 vs 打板：**

| 方式 | 操作 | 优缺点 |
|------|------|--------|
| **扫板** | 不等到涨停价就扫货，提前抢筹 | 确定性低但成本低 |
| **打板** | 最后一分钱才下单（如 9.99→10.00） | 确定性高，标准打板 |
| **排板** | 涨停价排队等成交 | 可能排不到 |

> "打板是小资金，半路/点火是大资金。没到那个体量不用学点火。" — 许文杰

**竞价引导/撤单：** 9:20 前可撤单→小资金制造假封单引导方向→最后几秒撤单。"最后 3 秒钟从 11 万手撤到 3700 手——很猥琐。"

**冲天炮：** 竞价尾买量 >> 尾卖量→开盘瞬间脉冲拉升。

**回首掏：** 早上换手不够炸板回落→下午换手充分后回封。"早上换手不够容易炸，下午换手充分后的回首掏比较有气质。"

**护盘辨别：** 国家队先动 50/300→护盘；1000 止跌反弹=真正止跌。只靠 50/300 拉=护盘而非真止跌。




### 8.9 分时翻红可能是卖点 ⚡ [v31 new]

一个反直觉的实战技巧：当你看到分时翻红时，可能是卖点而非买点。

> **许文杰原话：**「你们记得吧？翻红了说不定是个卖点。」 / 「你激动了、亢奋了、觉得他又行了的时候，提醒自己——说不定这个地方是卖点。」

**原理：** 弱势股的分时翻红往往是弱反弹，是持仓者减亏出局的机会，而非新资金进场的信号。

### 8.8 盘口卖点技巧 ⚡ [v30 new]

许文杰对盘口的实战用法（不依赖复杂指标，简单实用）：

| 盘口信号 | 操作 | 逻辑 |
|---------|------|------|
| 大单压在上方冲不上去 | 卖 | 抛压重，高点不过前高 |
| 大单托在下方不跌 | 持有/观察 | 有承接，但注意"托盘不涨=出货" |
| 扫单连续吃上方挂单 | 持有/可追 | 资金主动进攻 |
| 脉冲拉升后回落 | 卖 | 诱多，不是真拉升 |

**卖在尖尖的口诀：**
1. 盯盘口：一波拉高后，第二个高点来了
2. 看大单：上面有大单压着冲不上去
3. 执行：100 S 直接走，不犹豫

> "一波拉高之后，后面这个高点不过前面高点，盘口上面明显有点冲不上去——100 S 走掉就很容易烂在一个尖尖。" — 许文杰

**盘口心态：** 卖飞了没关系——"难道后面买不回来了吗？交易所又没封你账户。"

## 九、高低切与补涨

### 9.1 高低切判断

| 状态 | 现象 | 结论 |
|------|------|------|
| 真高低切 | 高位崩塌+低位载体持续走强 | 可参与 |
| **假高低切** | 低位载体 2 天死+换更低位 | ⚠️ 防守 |

### 9.2 高低切两种形式

**案例：** 建设工业高位横盘时，低位出现新时达等补涨。金奥博天地板退潮时，资金切换到消费方向低位。

1. **板块内高低切：** 同板块从高位切到低位补涨
2. **跨板块高低切：** 从老主线切到新低位板块

高度股崩塌 2-3 天后，低位新载体才能确立。

### 9.3 补涨体系

主升期做补涨（涨幅少的同板块票）。补涨高度受限于龙头。看 A 做 B：龙头超预期→低吸同属性后排套利。

### 9.4 二波体系

**二波风格隔离：** 二波票和单波票不在同属性内对比。二波票在二波风格内横向对比。

**N 波 vs 单波：** N 波票经历过一次分歧→穿越，抗跌性强。单波票首次分歧→可能结束。

**二波启动条件：** 一波结束后调整不破关键位（20 日线/前低）→放量启动→确认二波。

## 十、趋势股体系

### 10.1 趋势 vs 情绪节奏

趋势股：慢涨沿均线，高潮=尾声该卖。情绪股：快涨连板，高潮不一定结束。趋势买回调均线低吸，情绪买打板/半路。趋势持有沿均线，情绪断板即走。

### 10.2 趋势股高潮=尾声

趋势股沿均线慢涨→突然涨停潮→短线客看到才进场=已尾声。高潮日=该卖不该买。因为趋势股主力成本低，涨停潮=吸引接盘侠出货。与情绪股不同：情绪股高潮不一定结束，趋势股高潮=尾声。

### 10.3 一跌二锤三收涨

趋势股底部的经典 K 线形态——三步确认法：

> **许文杰原话：**「有一个口诀——一跌二锤三收涨。先跌一根阴线，然后第二天出锤子线（带长下影），第三天收阳收涨，基本就能确立止跌。」

**三步确认详解：**

| 步骤 | K 线形态 | 含义 |
|------|---------|------|
| 一跌 | 中阴/大阴线 | 空方释放 |
| 二锤 | 锤子线（长下影+小实体） | 空方衰竭，多方开始反击 |
| 三收涨 | 阳线收涨 | 确认止跌，买点出现 |

**失败判断：**
- 如果第三天不收阳而继续收阴→锤子线失效→止跌未确认
- "后面那一天还是收阴收跌，就说明那个锤子是失败的"

**实战应用：**
- 趋势股回踩时，等三天确认信号再动手
- 不要第一天跌就抄底，不要第二天锤子线就冲进去
- 第三天收阳后，回踩不破第三天的低点→加仓


### 10.4 1.5 天调整法则

趋势，这就操作策略这一块，趋势这一块的这一个步骤。趋势我做趋势股的步骤，我们前两天已经说过了，这个地方就简单的就过一下了。第一步其实就是你们也在做，我也给你们在做的成交盘的板块分类。第二步，根据每个板块每个分支的这个节奏，去找他们的二三定律，然后去找他们的一个调整两天震荡两天左右的一个低吸点，有时候1.5天，有时候两天，对吧？最后就是根据一些技术手段，去找他们的分时上的埋点，最终其实考验的是这个基本功。  

### 10.5 强趋技术

强趋势股的均线体系——许文杰的核心技术分析工具：

**均线优先级（短→长）：**
| 均线 | 用途 | 说明 |
|------|------|------|
| 5 天线 | 短线强弱分界 | 强势股回踩 5 天线即反弹 |
| 13 天线 | 中期支撑 | 5/13 金叉=趋势确认 |
| 15 分钟 MA45 | 超短线买卖点 | 量能＞2 万亿时更有效 |
| 30 分钟 MA45 | 短线支撑 | 正常市场的主要支撑 |
| 45 天线（日线） | 中长线趋势 | 机构票的生命线 |

**多周期配合：**
- 缩量行情（<1.5 万亿）→ 30 分钟 MA45 是强支撑
- 放量行情（>2 万亿）→ 15 分钟 MA45 就能撑住
- "最近强趋势股行情达到 15 分钟就下不去了，以前都得打到 30 分钟"

**强趋势的"不破"原则：**
- 强趋势股不会有效跌破 5 天线——破 5 天线 2 天以上=走弱
- 45 天线是中期最后防线——跌破=趋势结束
- 多次测试均线但始终不有效跌破=强

> **许文杰原话：**「15 分钟、30 分钟多看看。没有两万亿的市场，基本都会达到 30 分钟的 MA45。最近强趋势股行情达到 15 就下不去了。」


### 10.6 跌 30% 反弹定律

高位跌 30%→接近反弹。出阳线=反弹信号。但不适用于退潮初期的 A 杀。

### 10.7 反向验证法则 ⚡ [v26 new]

指数大跌 ≠ 趋势差。判断核心是**辨识度大票的反馈**，不是指数本身：

| 状态 | 特征 | 判断 | 案例 |
|------|------|------|------|
| **看似弱但挺强** | 指数阴包阳，但核心大票体面（微跌/收阳） | 正常调整，不 A 杀 | 5/14 中继新高+新易盛温和 |
| **看似强但挺弱** | 指数温和/收阳，但核心大票批量下跌 | ⚠️ 吹哨 / 毒苹果 | — |

**验证方法：看指数贡献度** — 谁在带指数跌？
- 二线板块（金融/有色/新能源）带指数跌 + 核心大票体面 → 正常调整
- 核心大票本身带指数大跌 → 真退潮信号

> "今天指数阴包阳，但中继新高、新易盛温和、天孚上涨——看似挺弱的但挺强的。反而如果今天指数小阴线但核心大票都大跌，那就是吹哨了。" — 5/14

### 10.8 次新割裂判断 ⚡ [v26 new]

次新内部科技 vs 非科技的日内轮动，反映资金真实态度（尤其在大票停牌/监管事件后）：

| 盘面现象 | 含义 |
|---------|------|
| 科技次新大跌 + 非科技次新大涨 | 热度在维系，科技分歧可控（上午 A 不行有 B） |
| 下午科技次新回流 + 非科技回落 | 资金没放弃科技，分歧后回补（下午 B 回落 A 起来） |
| 两者同时大跌 | ⚠️ 系统性风险，全面撤退 |

> "大普停牌后，连讯/盛合晶微-10%，但非科技次新（机器人/航天）冲+10%——热度维系着，分歧可控。" — 5/19

**运用：** 监管停牌事件 → 看次新内部是否割裂 → 割裂 = 可控，同跌 = 真恐慌。


### 10.9 技术指标的态度 ⚡ [v27 new]

许文杰对传统技术指标的态度是**否定的**，但也承认其博弈价值：

> "金叉死叉...我觉得好 low。技术指标这个东西没啥用。但是我以前走了很多弯路，就对技术指标肯定是有比较多的研究的。"

**唯一认可的简单用法：**

| 指标 | 用法 | 许文杰原话 |
|------|------|-----------|
| 五天线+三天线金叉 | 博弈点 | "五天线跟三天线的金叉点是一个博弈点" |
| 箱体 | 上沿压力/下沿支撑 | "箱体内部来回徘徊，下沿=最强走法" |
| MACD | 翘头后 A 字死叉 | "金叉好...但 MACD 会给你翘个头然后啪 A 死叉" |

**箱体操作：**
- 箱体下沿 + 量够 = 止跌，可低吸
- 箱体上沿 + 缩量 = 压力，可高抛
- 强势 = 下沿附近横住慢慢上；弱势 = 破下沿继续探

> 许文杰的核心观念：**技术指标是"纯数层面的东西"，真正的交易靠节点+辨识度，不是靠指标。**

## 十一、量能与成交量

### 11.1 量能分层锚定

| 量级 | 环境 | 策略 |
|------|------|------|
| < 1 万亿 | 冷清 | 空仓 |
| 1-1.3 万亿 | 弱势震荡 | 冰点修复 |
| 1.3-1.6 万亿 | 正常 | 可交易 |
| 1.6-2 万亿 | 活跃 | 积极参与 |
| > 2 万亿 | 爆发 | 容量主导 |

### 11.2 量能趋势监控

连续 3 日量能变化方向判断趋势：持续放量=增量资金入场；持续缩量=退潮；量能忽大忽小=震荡。

### 11.3 天量天价 / 地量地价

- **天量天价：** 天量后见天价（短期顶），天量次日不追。
- **地量地价：** 极致缩量后可能见底，需放量确认。

### 11.4 量价配合

| 组合 | 判断 |
|------|------|
| 放量上涨 | ✅ |
| 放量滞涨 | ⚠️ |
| 缩量上涨 | ✅ 锁仓 |
| 缩量下跌 | ❌ |

**下午补量：** 下午<上午预估值→拉高持续不了。放量滞涨→"偏震荡，不会连续拔高。"



### 11.6 爆量出妖股 ⚡ [v31 new]

爆量（成交量暴增）是龙头/妖股的必要但不充分条件：

> **许文杰原话：**「爆量出妖股。妖股一定会受到爆量的检验。但不是说所有爆量都会走出来——妖股要走出来，龙头要走出来，一定会经历爆量。这是一个充分非必要条件：妖股一定会经历爆量，但爆量的票不一定都是妖股。」

**爆量分类：**
1. **分歧日爆量**（最佳）——市场本身分歧，该爆的充分爆，扛住的才走得出来
2. **拉升中爆量**——次之，筹码换手充分
3. **高位滞涨爆量**——危险，可能是出货

> **许文杰偏好：**「我更加喜欢在分歧日那天，本身市场就是分歧那天，那种爆量能扛住的板。那一天可以锁定目标。」

### 11.7 充分非必要条件 ⚡ [v31 new]

许文杰交易哲学的核心思维工具——**没有 100%，只有概率**：

> **许文杰原话：**「不是说加速一定死。只是每一次加速大概率脱离不了这两种情况。但出现这两种情况不代表一定死——这是一个充分非必要条件，没有 100% 的事情。起码来说出现这种现象了，值得留个心。」

**充分非必要条件** = 现象 A 出现→大概率出结果 B，但 B 也可以从其他路径产生。A 是 B 的充分条件但不是必要条件。

**在交易中的应用：** 每个信号都是概率提醒，不是绝对判定。留个心≠一定走。

### 11.5 爆量体系

> 📎 参见: [11.6 爆量出妖股], [11.7 充分非必要条件], [5.11 地天板]

## 十二、仓位管理与防守



### 12.1.1 仓位控管决定心态 ⚡ [v31 new]

仓位管理的本质不是防亏损，而是**保持心态**：

> **许文杰原话：**「仓位控管决定了心态。什么时候该做什么样的事，什么时候该控制什么样的仓——不是说不能做，而是要用仓位去平衡。」

**仓位与节点对应：**
| 节点类型 | 仓位 | 心态 |
|----------|------|------|
| 冰点转折/启动节点 | 重仓（7-9成） | 自信 |
| 上升中段 | 中等（5-7成） | 从容 |
| 混沌期 | 轻仓（2-5成） | 试探 |
| 退潮期 | 空仓或微仓 | 防守 |

> **许文杰原话：**「很多时候我们规避掉风险，不是很刻意的说'我这个地方要空仓'——是这个地方正好没有你的买点，自然而然没有去。市场调了你就规避了。」

### 12.1 仓位管理

| 周期 | 仓位 |
|------|------|
| 冰点 | 0-20% |
| 启动 | 30-50%，确认后加 |
| 主升 | 80-100%，满仓单调 |
| 高位震荡 | 30-50% |
| 退潮 | 0-10% |

### 12.2 战术/战略分离

- **战术层面：** 日内买卖点、仓位微调
- **战略层面：** 周期阶段判断、大方向定调
- 战术服从战略：周期是退潮→战术再好也不出手

### 12.3 两种防守方式

1. **主动防守：** 降仓→空仓，管住手，等节点
2. **被动防守：** 持仓中找性价比位置出，不恋战。"退一退返一返"的温和退中可用。

### 12.4 红线禁忌

**案例：** 退潮期做弱转强→金奥博 D4 弱转强被套。亏损加仓→建设工业卖飞追回被二次伤害。后排格局→跟风票板块退潮时跌最惨。

退潮期五大忌：①不接加速 ②不抄底（退潮期反包大概率失败）③不做中位（跌最惨）④不幻想穿越 ⑤不等节点就出手。核心原则：退潮期不亏钱>赚钱，控制回撤比进攻重要100倍。

### 12.5 负反馈锚定法

| 锚定类型 | 观察要点 |
|---------|---------|
| 抱团强势股 | 竞价是否温和、低开幅度 |
| 连板断板股 | 是否继续大跌 |
| 昨日跌停股 | 跌停是否收窄 |

**止跌 K 线组合：** "两阴一阳再收阴收下影线→慢慢止跌，即使还在创新低。"

### 12.6 主动走弱 vs 被动走弱

| 类型 | 含义 | 操作 |
|------|------|------|
| **主动走弱** | 不跟指数/板块，自己跳水 | 真的弱，不是低点 |
| **被动走弱** | 被指数/板块拖累 | 可能情绪低点 |

> "龙头主动申请跌停=真弱。被冤枉带下来=情绪低点。"

### 12.7 预警体系

异动公告=检讨书。发完检讨书→资金短期收敛→随后纠偏→可能纠出双头或新高。核心：①别在异动公告日恐慌 ②别在异动后第一天追 ③等资金纠偏后再判断方向。

### 12.8 止涨震荡操作细则 ⚡ [v26 new]

止涨震荡（涨不动但跌不深，量还够）阶段的实操准则：

| 动作 | 允许？ | 说明 |
|------|--------|------|
| **新开仓** | ❌ | 不放新开，不盲目开超市 |
| **做 T** | ✅ | 冰点附近低吸→拉高 T 出，2-3 个点就满足 |
| **拉高了结** | ✅ | 逢反抽减仓，落袋为安 |
| **持有强区** | ✅ | 强区体面可多拿一天，反正止跌时会给你从容出场 |
| **强更强追高** | ❌ | 止涨期不追 |

> "我不太敢也不太想新开了。要么做 T，要么拉高了结。往下冰点做 T 很稳妥——万一尾盘又杀，你新开的又被套。" — 5/14

**仓位配合：** 指数冲不动时，仓位不能像主升期那么大，稍微收一点做。量跌破 3 万亿再加防守。

## 十四、纪律与心态

### 14.1 核心纪律

> "交易是跟自己去比。在自己能力圈里做到最好。"

> "市场才是爸爸。跟随+应对 > 主观预判。"

> "任何一笔 B/S 都要明明白白。亏得明白，赚得明白。"

> "学会取舍，往阻力小的方向走。不要什么都买——最后账户变超市。"

### 14.2 先手思维

有持仓=有先手=有选择权。退潮期先手=跑得比别人快。先手不是追高的理由。核心：买在先，卖在先，永远比市场快半步。

### 14.3 主力行为观察

通过大单、托单、压单判断主力意图：托盘但价格不涨=出货；压盘但价格不跌=吸筹。竞价引导：小资金制造假封单→最后几秒撤单=猥琐操作。

### 14.4 资金合力

多路资金（游资+机构+散户）同时看好的方向=合力方向。判断方法：涨停板上不同席位接力、机构和游资共同参与、不是一家独大。合力方向=最可持续，单一资金主导=走不远。

### 14.5 45 天线

45 天线是许文杰最看重的长周期趋势线——用于判断个股/指数是否还处于上升趋势：

> **许文杰原话：**「45 天线——英伟达的回踩正好在 45 天线。周线层面也正好在这。」

**45 天线的实战意义：**
- 45 天≈9 周≈2 个月——中长线资金的持仓周期
- 趋势股在 45 天线上方运行=健康
- 跌破 45 天线→趋势可能结束，不可抄底
- "破 45 天了"=趋势票判死刑

**45 天线与量能：**
- 量能充足→回踩 45 天线即反弹
- 量能萎缩+破 45 天线→大概率继续跌
- 配合周线看——45 天线和 10 周线往往位置接近

**多种场景：**
- 上升趋势中回踩 45 天线→买点
- 下降趋势中反弹触 45 天线→卖点
- 45 天线走平→震荡，可高抛低吸
- 45 天线向下拐→空头趋势，远离


### 14.6 异动后"检讨书"规律

异动公告后→资金短期收敛→随后纠偏→纠出双头或新高。别在异动公告日恐慌。

### 14.7 止损纪律 ⚡ [v30 new]

> "你们不能够说做股票总是能接受赚钱，不接受止损。进去的时候想好了止损线，实操中就变成'我只接受你给我过压力线，我不接受你给我止损破五天线'。他凭啥一定要朝着你的想法去？"

**止损铁律：**

| 原则 | 内容 |
|------|------|
| **进场前设好** | 买入时必须想好止损线（五天线/三天线/前低） |
| **执行不打折** | 到了止损线就走，不讨价还价 |
| **不扛单** | "市场才是爸爸"，不对抗趋势 |
| **止损后不追** | 止损了就是错了，不急着翻本 |

**胜率/直博率思维：**
- 冰点日最强标 + 第二天市场修复 = 高直博率（值得博弈的胜率）
- 不是每笔都能赚，但要在高胜率节点出手
- "如何把胜率提高？回到四位一体。"

> 许文杰原话："市场不管怎么走，我都有自己的离场条件，我都能够全身而退。"


### 14.8 踏空与追高 ⚡ [v30 new]

> "踏空资金会去追后排，但从审美角度不会盲目做后排。前排没机会就等分歧。"

**踏空应对：**

| 错误做法 | 正确做法 |
|---------|---------|
| 踏空后立即追高后排 | 等分歧，分歧后前排可能给机会 |
| 怕错过强行上车 | "该板的都板了，再追后排我不愿意" |
| 踏空后焦虑乱买 | 宁可错过，不做错 |
| 看别人赚钱眼红 | "每一轮行情真正带走利润的不超过20%" |

**核心心态：** 前排没机会就等分歧→分歧后做前排。"今天前排买不到，你只能追高后排。拉长时间来看，追高后排的事不做。"

**许文杰的审美：** "我宁可错过前排也不做后排。踏空资金去做套利我可以理解，但从审美角度看不上。"


### 14.9 知行合一 ⚡ [v30 new]

> "知行合一，说说简单，真的做起来没那么简单。"

**知行不合一的根源：**

| 现象 | 本质 |
|------|------|
| 明明判断该卖但不卖 | 侥幸心理——"万一又起来了呢" |
| 明明判断该买但不买 | 恐惧——"万一买完就跌呢" |
| 盘中临时改预案 | 被盘面情绪绑架 |
| 做 T 变加仓 | 忘了"每笔交易是独立的" |

**许文杰的解决方案：**
1. **每笔交易独立：** 做 T 前先问自己——"如果我是空仓，我会不会买这个票？" 不会→不做 T
2. **遵守预案：** 前一天写的预案，第二天不要因为盘面波动全推翻
3. **问自己：** 你是来赚钱的，还是来享受交易快感的？如果是后者，去坐过山车

> "早上看到了正反馈+分单及格，按理来说是开仓点。事后说'不该开仓'就是马后炮。知行合一是在当时条件满足时果断执行。"

## 十五、关键概念速查

**K 线形态：** 白织带（开盘=最低，极端上涨）/黑织带（开盘=最高，极端下跌）/吊颈线（长下影+小实体，顶部反转）/十字星（变盘）/揉搓线（洗盘）/T字板（分歧转一致）

**身位优势：** 先涨停的对后涨停的有身位优势。同属性横向对比，先板的身位更高。

**压制与突破：** 前高是压制，放量突破=买点。突破后回踩前高不破=确认。

**资金就近原则：** 板块轮动按产业链上下游/属性扩散推进。从核心扩散到边缘。

**该强不强 / 该弱不弱（预期差博弈）：** 预期强但实际弱=卖点；预期弱但实际强=买点。

**看 A 做 B：** 龙头超预期→低吸同属性后排套利。但只能在板块上升期做。

**消息处理：** 利好不涨=利空；利空不跌=利好。消息驱动高开→市场认（回封）=合力。

**理想交易：** 买在分歧转一致的节点→持有到一致转分歧→卖出。不追求完美，追求大概率。

### 5.9 竞价看什么 ⚡ [v32 new]

竞价 9:15-9:25 是全天最重要的 10 分钟。许文杰竞价的关注点：

**三个核心维度：**
1. **高标反馈**：昨天的高标竞价涨跌？跌停封单有多少？
2. **核心板块竞价**：主线板块的龙头竞价强弱？有无抢筹？
3. **竞价量能**：竞价成交占昨日成交的比例（>4% 属于爆量竞价）

**竞价态度：**
- 竞价高开≠超预期（周末消息刺激的高开要打折）
- 多看几秒钟——竞价瞬间抢筹可能不稳，等开盘确认
- 9:45 前基本能判断当天方向

> **许文杰原话：**「周末有消息刺激，竞价高开不算超预期。竞价时抢筹，我的习惯是多看几秒钟或一两分钟——竞价去的话有点偏赌。」

**竞价实战优先级：**
| 竞价信号 | 可信度 | 操作 |
|---------|--------|------|
| 无消息刺激+爆量高开 | 高 | 重点跟踪 |
| 有消息刺激+高开 | 中 | 等开盘确认 |
| 竞价跌停封单大 | 高（负面） | 当天回避该方向 |
| 竞价抢筹+高开 | 中 | 多看一分钟 |

### 5.10 炸板回封 ⚡ [v32 new]

炸板（涨停打开）后能否回封，是判断个股强弱的关键指标：

**回封强弱判断：**
| 情况 | 强度 | 含义 |
|------|------|------|
| 瞬间炸开→秒回封 | 最强 | 筹码稳，只是被大盘带开的 |
| 炸板跳水→拉回封板 | 中等 | 有分歧但多方胜 |
| 炸板→震荡→尾盘回封 | 偏弱 | 勉强封住，次日预期低 |
| 炸板→不回封 | 弱 | 次日大概率低开 |

**主动性判断：**
- 谁先炸→谁弱；谁先回封→谁强
- 一只股炸板带崩其他→它是锚定；其他炸板带崩它→它是被动

> **许文杰原话（分析海德 vs 电光）：**「上午是电光更强，下午是海德更强——上午跟下午角色完全相反。从延续性角度来说，海德更占优。」

**回封与板块联动：**
- 某个大票（如联通）强→带动小票回封→板块持续性确认
- 无人带动独自回封→次日关注弱转强

### 5.11 地天板 ⚡ [v32 new]

地天板（当日从跌停拉到涨停）是辨识度的超级加分项，但有严格条件：

**地天板的三层含义：**
1. **极致分歧后的共识**：从跌停到涨停，说明多空剧烈博弈后多方完胜
2. **辨识度碾压**：全天所有股票中最吸睛的一只
3. **次日大概率弱转强**：地天板次日往往有加速

**案例：大千生态 19 号地天板→20 号弱转强加速**

> **许文杰原话：**「大千 19 号那一天平开跳跌停，摸跌停，地天板、烂板、全天。第二天加速弱转强。」

**地天板的实战要点：**
- 必须是换手充分的地天板（烂板>秒拉）
- 地天板当日是买点（低吸），次日加速是加仓/持有
- 如果地天板后次日不加速→低于预期，考虑减仓
- 不是所有地天板都能走出来——同一个周期里只有 1-2 只能成

### 5.12 龙回头 ⚡ [v32 new]

龙回头（老龙头超跌后的反弹）是一种模式，但许文杰的态度很明确：

> **许文杰原话：**「俗称龙回头，结果就是占龙头便宜。做辨识度票做龙头做久了，你是看不上这种的。」

**许文杰为什么不怎么做龙回头：**
- 龙回头本质是「占便宜」心态——想捡便宜货
- 老龙头已过主升期，反弹空间有限
- 新周期有新龙头，何必回看旧龙？

**龙回头的适用场景（如果非要做）：**
- 老龙调整幅度 30%+ 且缩量止跌
- 新周期确立后，老龙作为补涨反弹
- 只能低吸，不能追高，预期 10-15% 就走

### 5.13 分时承接的正确态度 ⚡ [v32 new]

很多交易者过分关注分时承接，但许文杰认为层次要高于它：

> **许文杰原话：**「我要的是那个时机，而不是说你有多强我就会来。不是说看你分时有承接再换手。你要有大局观，不能只有个股思维，要有板块思维、体系思维、节点思维。当大环境不支持你的时候，你要学会舍弃。」

**分时承接的合理用法：**
- 只在节点正确、板块正确的前提下，才看分时承接来选具体买点
- 脱离节点和板块看承接=舍本逐末
- 尾盘承接（尾盘未跌停且有人接）→说明有资金博弈次日修复

**承接判断优先级：** 节点 > 板块 > 个股辨识度 > 分时承接

### 14.11 开盘即下班 ⚡ [v33 new]

"开盘即下班"是一种极致的交易状态——开盘后短时间内即完成全天操作：

**含义：** 开盘 15-30 分钟内，你已确认当天所有持仓的去留。剩下的时间只是验证，不再操作。

**适用条件：**
- 上升期/强混沌期，方向明确
- 持仓都是前排核心
- 盘前预案充分，各路径都想清楚了

> **许文杰原话（秒板）：**「海德开的比均和高，竞价就弱转强，然后秒板。它的涨停带动着算力、顺纳、中恒——开盘就结束了。」

**秒板判断：**
- 秒板+缩量→强（锁筹成功）
- 连续两天缩量秒板→危险（获利盘累积，第三日容易炸）

### 14.12 开板买入 ⚡ [v33 new]

开板（涨停打开）的买入逻辑——许文杰特定场景下的策略：

**允许开板买入的场景：**
1. **冰点转折日**：指数/情绪冰点后，核心股炸板→是市场的恐慌，不是个股的问题
2. **尾盘捡漏**：博弈次日修复，等抱团股开板捡漏

> **许文杰原话：**「今天指数给到的分歧，再怎么样硬着头皮也该去的——明天即便被蒙了，也是模式内的比例。」

**禁止开板买入的场景：**
- 已经加速过的股炸板（"再次开板上车的时候说明退潮了"）
- 高位横盘后炸板（出货而非分歧）
- 退潮期炸板（接飞刀）

**核心原则：** 开板买入=赌分歧后的修复。只在上行周期用，退潮期绝不用。

### 14.13 回踩确认 ⚡ [v33 new]

回踩确认是趋势股的经典买点——不是所有的回踩都要买，要等确认信号：

> **许文杰原话：**「真正把线突破了，再回踩的时候——缩量小K + 五天线——加起来好买点。有时候还要配合 30 分钟图和 60 分钟图。」

**回踩确认三步：**
1. **突破**：股价突破关键均线/压力位
2. **回踩**：回调到突破位附近，缩量，小 K 线（不恐慌）
3. **确认**：回踩不破 + 再次放量上攻 → 买点

**金叉回踩：** 5 天线和 13 天线金叉→回踩金叉点附近→买点。

**纪律：** 如果回踩破了关键位→放弃，不赌。那么多标的，不差这一个。

### 14.14 倒金字塔加仓 ⚡ [v33 new]

许文杰不推荐散户倒金字塔加仓（越涨越加），正确的仓位节奏是：

**正确加仓节奏：**
| 阶段 | 仓位 | 逻辑 |
|------|------|------|
| 冰点/启动（不确定期） | 小仓试错（1-2成） | 验证判断 |
| 确认上升（分歧后继续强） | 加仓（3-5成） | 信号确认 |
| 主升加速 | 持有不追加 | 享受利润 |
| 高潮/一致 | 减仓 | 锁定利润 |

**三不原则：**
- 不一次重仓（先试错再确认）
- 不在加速后加仓（高位加仓=一把亏光）
- 不在退潮时补仓（亏损加仓=错上加错）

### 14.15 割肉纪律 ⚡ [v33 new]

许文杰对止损的核心态度——不是怕亏钱，是怕亏得不明不白：

> **许文杰原话：**「以前是浑浑噩噩的亏，现在可能会有止损、有亏钱。但你明明白白——虽然都是亏钱，差距大了去了。因为你现在知道原因，以前是没有任何章法。」

**割肉三原则：**
1. **破位必割**：破均线/破关键支撑 → 走
2. **风格不对必割**：情绪票用趋势手法持有 → 走（情绪票破位=割在跌停也要割）
3. **节点不对必割**：退潮期开仓+亏损 → 立即走，不格局

**心态：** "亏 10-20 个点，从内心角度能接受，从其他地方都能回来。但亏了再多就不合适了——后面都是随机性了。"

### 14.16 聚焦盘面 ⚡ [v33 new]

许文杰反复强调的核心心法——两耳不闻窗外事：

> **许文杰原话：**「沪上开跑车那个事儿，我也是周五收盘之后才知道的。虽然加了很多超跑群，但都没时间看。两耳不闻窗外事，兄弟们。」

**"聚焦盘面"四字原则：**
- 不看微博/抖音/公众号的情绪发酵
- 不被周末的消息面绑架判断
- 不研究宏观/国际/基本面——"中线的研究范围太广了，你得研究宏观、国内、国外"
- 只做一件事：看盘面本身的信号

**周末消息的处理：**
- 周末暴吹的题材→周一竞价高开≠超预期→等开盘验证
- "被舆情牵着鼻子走，累不累？聚焦盘面本身就可以了。"

> **许文杰原话：**「我就说指数 OK 没问题，有消息刺激挺强的，下午也逐步回落了——为什么？因为情绪本身并不好。消息改变不了情绪周期。」

### 14.17 历史类比法："今天像哪一天" ⚡ [v34 new]

许文杰盘中/复盘时最常用的一个思维习惯——把今天的行情类比到历史某一天：

> **许文杰原话：**「你们觉得今天这个节点有点像什么时候？早盘就是——不是闲聊，是真的一会儿也会讲到的内容。」

**历史类比的价值：**
- 股市虽不简单重复，但节奏高度相似
- 类比历史能快速判断当前处于什么阶段
- 知道"上次这样走之后发生了什么"→应对有谱

**典型类比案例（许文杰实战）：**
| 当前行情 | 类比 | 含义 |
|---------|------|------|
| 高标断板+次日跌停 | 去年 9.9 深圳华强 | 退潮确认，谨慎 |
| 拓维日K | 10.25 长山 | 量能不同，类比仅供参考 |
| 机器人内部轮动 | 柴发（华丰/泰豪/潍柴） | 高位三傻内卷 |
| 一字未开+分歧不够 | — | 假修复，还会再砸 |

**用法：** 每天问自己——"今天像哪一天？上次这样走之后发生了什么？"

### 14.18 做笔记与看回放 ⚡ [v34 new]

许文杰反复强调——交易能力的提升，依赖盘后的深度加工：

> **许文杰原话：**「真的看回放有用的。你平时多花时间——盘中的内容多看多想，盘后直播内容多思考、多做笔记。需要用量变到质变。一周两周不行，一个月两个月；一个月两个月不行，一个季度两个季度——你悟性再差半年一年总归有变化。」

**许文杰式笔记法：**
1. **当场记核心**：盘中群消息+直播内容，记关键词和结论
2. **盘后整理**：对照自己的持仓/操作，写下"为什么做对了/做错了"
3. **回放验证**：看回放时对照当时的判断→事后验证→修正认知

**笔记的重点不是"记了什么"，而是"想了什么"：**
- 为什么当时觉得是买点？
- 为什么没看到那个风险？
- 如果再来一次会不会做同样的决定？

> **许文杰原话：**「你们盘后做笔记啥的——其实我已经帮你们做完一半了。」

### 14.19 亏钱两大源头 ⚡ [v34 new]

许文杰总结短线交易亏损最大的两件事：

> **许文杰原话：**「杂毛 + 年票。做杂毛（后排、非核心）和做年票（格局、不卖），是短线资金亏损最大的两件事。」

**杂毛（后排/非核心）：**
- 贪便宜买后排→板块一退潮最先跌
- 后排没有辨识度→没有接盘资金
- 解决：只做第一梯队，横向对比选最强

**年票（格局/不卖）：**
- 趋势走坏还持仓→"再看看"→越亏越多
- 短线做成中线→中线做成股东
- 解决：模式内卖点触发→立即执行，不幻想

> **许文杰原话：**「千万不要把短线做成中线。短线就是短线，该走走。模式内的卖点不能保证卖在最高，但保证卖得明明白白。不年票不亏大钱。」

### 14.20 操作时间窗口很窄 ⚡ [v34 new]

许文杰揭示的残酷真相——真正能赚钱的窗口非常短暂：

> **许文杰原话：**「这个阶段的操作时间很狭窄。你想这一波上升期就维持了三四天，真正能够持股的有植入空间的可能就一两天。甚至窄一点可能半天到一天。」

**时间窗口的实战含义：**
- 一个月可能只有 3-5 天是确定性的赚钱窗口
- 其他时间要么混沌要么退潮→不亏就是赚
- 错过了窗口期→不追→等下一个窗口

**心态调整：**
- 不要因为"今天没交易"焦虑——没买点就是没买点
- "很多时候我们规避风险，不是刻意空仓——是正好没有你的买点"
- 一年抓住 3-5 波确定性大的行情=超额收益

### 14.21 卖点后置 ⚡ [v34 new]

与"买点后置"对称——在特定市场环境下，卖点也要后置：

> **许文杰原话（银之杰案例）：**「你昨天尾盘加仓的人，今天银之前卖在跌停也没错——为什么？跟天风一样，卖点后置。银之前是整个市场最高标最核心的品种，你多格局一下怎么了？」

**卖点后置适用条件：**
| 条件 | 说明 |
|------|------|
| 持仓是市场最高标/最核心 | 辨识度最高的票，多给一点时间 |
| 上升期/主升浪中 | 趋势还在，回调可能是洗盘 |
| 有足够的利润垫 | "之前已经有足够的利润了" |
| 板块还在续强 | 锚定板块没走弱 |

**卖点后置≠不卖：**
- 核心票→多格局半天到一天
- 后排票→该走就走，不后置
- 退潮期→不后置，立即走

### 14.22 情绪传导链 ⚡ [v34 new]

高位股的情绪会传导到中位、低位，形成链式反应：

**传导路径：**
```
高位崩塌 → 中位补跌 → 低位连带 → 全市场冰点
     ↓
  资金从高位撤离 → 流入低位新方向 → 高低切换
```

**传导的判断要点：**
- 高位股跌停→看中位是否跟跌→跟跌=退潮确认
- 高位横住→中位有机会穿越→低位可补涨
- 老周期高标一直跌→压制新周期高度

> **许文杰原话：**「日出一直跌一直大跌的话，会反过来影响新周期的高度。老周期一直没止跌，新周期的高标会受影响。」

**传导锚定法：**
- 每天先看最高标（龙头）→再看中位→再看低位
- 连锁跌停=系统性退潮→全面防守
- 高位跌但低位不跟→可能只是高低切，不是退潮

### 14.24 牛熊策略差异 ⚡ [v34 new]

许文杰对牛市和熊市的实战区分：

| 维度 | 牛市 | 熊市 |
|------|------|------|
| 退潮方式 | 肉面共存，温和退 | 集体崩塌，A杀 |

- 冰点修复快但退潮也快

## 20. 深度行情复盘案例 ⚡ [v36 new]

以下为许文杰重点复盘的历史交易日，每段包含完整的行情背景、操作逻辑和教训。

### 20.1 2024.12.31 — 跨年博弈的典型日

**背景：** 12 月最后一个交易日，市场对「跨年龙」有强烈预期。
**盘面：** 高标异动，资金博弈跨年后的溢价。
**许文杰判断：** 不确定昨天的强势是不是为了跨年效应，需要延后一天确认。

> **关键原话：**「如果说昨天是普普通通的一天，那确实可以作为 10cm 打出高度之后的节点。但因为我不确定昨天是不是为了跨年，所以得延后一天看。」

**教训：** 特殊日期（节前/年末）的行情需要打折扣——可能是资金做账/博弈节日效应，不能当正常节点判断。

### 20.2 2025.1.2 — 电光科技地天板日

**背景：** 节后第一个交易日，市场在退潮后寻找修复载体。
**盘面：** 电光科技走地天板，建设工业竞价高开更主动。
**许文杰判断：** 横向对比电光 vs 建设，建设更主动（竞价更高、开盘更强）。电光的地天板虽然吸睛，但不确定能否扛住。

> **关键原话：**「横向对比来说，电光当时是地天，不确定能不能扛得住。建设竞价开得比中百更高，更主动。」

**教训：** 地天板=辨识度加分，但不等于次日一定强。横向对比谁更主动才是关键。

### 20.3 2025.1月下旬 — 金奥博退潮案例

**背景：** 金奥博 1/23 天地板跌停，冀凯股份跟随。
**许文杰判断：** 退潮从 1/23 开始。退潮三天后的修复，应该在 D3 尾盘拿先手（低吸），而非 D4 拉升后追。

> **关键原话：**「退潮三天之后的修复，不如昨天尾盘拿。今天看到拉升再去，就有点后手了。」

**教训：** 退潮期博弈修复的正确姿势——在退潮末端（D3尾盘）低吸，而非看到修复信号后再追。

### 20.4 2025.3月 — 机器人三傻内卷

**背景：** 机器人板块高位横盘，华丰/泰豪/潍柴（10cm高位三傻）互相争夺。
**许文杰判断：** 类似之前柴发板块的内卷——高位股之间互相卡位、争夺地位。做多错多，不如等方向明确。

> **关键原话：**「有点像当时柴发里面的华丰泰豪跟潍柴一样——高位三傻，不断内卷，不断争夺。」

**教训：** 高位多股争霸=消耗行情。选不出绝对龙头时，休息比强行选一个更明智。

### 20.5 2025.2.14 — 新炬网络「管住手」日

**背景：** 新炬网络当日表现类似之前的某个高位崩塌日。
**许文杰判断：** 回过头看，那天管住手确实有必要。

> **关键原话：**「回过头来看，你们是不是觉得新炬那一天我们管一管手，确实是有必要的？」

## 22. 许文杰式每日复盘模板 ⚡ [v41 new]

### 复盘九步（详细版）

```
第一步：看指数（2 分钟）
  □ 上证/创业板/中证 1000 涨跌
  □ 处于 5 日线上方还是下方？
  □ 指数与情绪的背离情况？
  → 结论：指数是【上升/震荡/下跌】

第二步：看情绪（3 分钟）
  □ 昨日高标今天竞价/收盘怎么样？反包了还是跌停了？
  □ 跌停家数？有没有核按钮？
  □ 上涨家数？涨停家数？
  □ 现在是周期哪个阶段？（冰点/启动/上升/高潮/分歧/退潮）
  → 结论：情绪是【上升/混沌/退潮】

第三步：看量能（1 分钟）
  □ 两市成交额 vs 昨日
  □ 量能趋势（在放大还是缩小？）
  □ 哪个板块成交最大？
  → 结论：量能【充足/正常/萎缩】

第四步：定环境（1 分钟）
  □ 结合前三步，今天处于什么环境？
  □ 对应的仓位策略是？
  → 结论：环境【好/一般/差】，仓位【重/中/轻/空】

第五步：看板块（5 分钟）
  □ 今天哪些板块涨了？（按涨停家数排）
  □ 哪些板块调了？（符合 23 定律吗？）
  □ 主线是谁？有没有新方向？
  □ 每个板块的梯队完整度？
  → 结论：主线【___】，关注【___】

第六步：看个股（5 分钟）
  □ 各板块的龙头是谁？（连板/容量/反包分别看）
  □ 哪些是共振梯队（二板日）的票？
  □ 有没有弱转强/地天板/炸板回封的票？
  □ 横向对比：谁的辨识度最高？
  → 结论：核心关注【___】

第七步：看风格（3 分钟）
  □ 今天趋势强还是情绪强？
  □ 大盘股还是小盘股领涨？
  □ 10cm/20cm/30cm 哪个最强？
  → 结论：风格偏向【___】

第八步：写预案（10 分钟）
  □ 明天大概率走什么路径？（先砸后修/先冲后杀/单边）
  □ 如果走出路径 A→做什么？
  □ 如果走出路径 B→做什么？
  □ 明确明天的【可做】和【不可做】
  → 结论：明日预案已定

第九步：对照持仓（5 分钟）
  □ 当前持仓在明天预案中的位置？
  □ 哪些需要持有/减仓/清仓？
  □ 哪些新的可以加自选？
  → 结论：持仓调整计划
```

## 23. 许文杰直播/复盘结构分析 ⚡ [v42 new]

理解许文杰每天的直播内容结构，能帮你更快找到重点：

### 每日直播五段论

| 段落 | 时间 | 内容 | 关键词 |
|------|------|------|--------|
| 1. 大势定调 | 前 5 分钟 | 今天的指数+情绪+量能总结 | "先定大势""面怎么样" |
| 2. 周期位置 | 5-15 分钟 | 现在处于周期的哪个阶段？与历史哪天像？ | "现在是混沌/退潮/上升""像哪一天" |
| 3. 板块逐个 | 15-40 分钟 | 主要板块逐个分析——谁强谁弱、预期差在哪 | "XX板块今天…""明天看XX" |
| 4. 个股/答疑 | 40-60 分钟 | 回答群友提问、重点个股点评 | "有同学问XX…""我说过XX" |
| 5. 预案总结 | 最后 10 分钟 | 明天的几种路径+操作预案 | "明天大概率…""如果…就…" |

### 许文杰直播的 5 个高频句式

| 句式 | 含义 | 何时出现 |
|------|------|---------|
| "我说一下…" | 下面是重要内容 | 每个新话题的开始 |
| "你们有没有发现…" | 引导独立思考 | 讲方法/规律时 |
| "我不知道你们记不记得…" | 复习之前讲过的重要内容 | 讲方法/规律时 |
| "这句话我说过很多遍了…" | 这是核心原则，必须记住 | 强调纪律时 |
| "这个地方就一句话…" | 最精炼的总结 | 复杂分析后 |

### 许文杰的内容密度规律

- **周一至周三**：内容最密集（分析周末消息+新的周期方向）
- **周四**：中等密度（周期中段确认/修正）
- **周五**：偏总结+下周预案（博弈周末消息）
- **节前最后一天**：偏谨慎（不轻易开新仓，防节日效应）

### 如何高效听许文杰直播

1. **先看结论**：他会在开头和结尾给核心判断
2. **再看板块**：每个板块的定性（主升/分歧/退潮）
3. **最后看个股**：个股分析是验证你对板块判断的
4. **记关键词**：不是记每句话，是记"节点""退潮""卡位""超预期"这些关键词
